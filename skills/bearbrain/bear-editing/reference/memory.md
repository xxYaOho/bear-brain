# Memory 模板

## 定位
`#memory` 是长期经验层，用于沉淀可复用知识、方法和判断依据。

## 主文件结构

```md
# #memory

## Position
长期经验入口

## Core Memory
- <可复用的核心原则、方法、约束>

## Recall Keys
- <检索关键词 -> [[memory/topic]]>

## Related Notes
- [[memory/topic]]
  - `NOTE-ID`
  - 主题子笔记
```


## 写入规则

1. 只有经过 promote 提炼的内容才能进入 memory
2. memory 强调精炼、可复用、可追溯
3. 定期回顾，合并相似主题，删除过时内容
